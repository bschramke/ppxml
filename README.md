# ppxml
A simple Java library to read [Portfolio Performance](http://www.portfolio-performance.info) XML files.

## Prerequisites
* [Java 8](http://www.java.com)

## Build it
```bash
git clone https://github.com/bschramke/ppxml.git
cd ppxml
./gradlew clean build
```
